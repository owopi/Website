    Advantages of Linux
    Open Source Parrot OS is a free and open-source Linux distribution based on Debian. It appeals to security professionals, ethical hackers, developers and those concerned for their online privacy.
    Security Implemented Parrot OS includes a full portable arsenal for IT security, penetration testing, and digital forensics operations. It also includes everything you need to develop your own programs or protect your privacy while surfing the net.
    Desktop Environment The operating system ships with either MATE, KDE or XFCE (only Home Edition) Desktop Environments.

    Benefits

    Pre-installed QOL Application
    App Name: Anonsurf
    Anonsurf is a tool made by Parrot OS developers that makes your online activity almost entirely anonymous. Not only does Anonsurf route all your traffic through Tor, but it also lets you start I2P services and clear any traces left on the user disk. Anonsurf also uses a Pandora Bomb to automatically overwrite the RAM when the system is shutting down. The best part is that all this is contained in a simple start/stop function.
    App Name: WifiPhisher
    WifiPhisher is a Wi-Fi security tool penetration testers use for testing network security. WifiPhisher can achieve a man-in-the-middle position against devices connected to a network by performing targeted Wi-Fi association attacks. Wifiphisher can also be used to set up a customized phishing attack against the connected clients to capture credentials or inject malware.
    Summary Parrot OS comes preinstalled with several valuable tools and programs such as AnonSurf, many compilers and IDEs, and hacking tools like WifiPhisher.
