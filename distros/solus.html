    My Personal Favourite!

    What is Solus OS?
    Solus OS It is similar to Arch Linux, being built from bottom to top! You get to build everything from scratch just like Arch!
    So why Solus and not Arch? Solus OS focuses on aesthetics and user-friendly GUI, unlike Arch. There is no hassling with settings or choices, as almost everything is taken care of for you.
    Are there versions or updates? There are not! What are newer updates? Basically going from windows 10 to windows 11! What are newer versions? Ubuntu 18.04 to Ubuntu 20.04!
    How would I get the Newest Solos OS? Solus OS is also a rolling release distribution, making it similar in functionality to Arch Linux. This is beneficial as you will only have to install Solus OS once and never update to a new version (for example, Windows 10 to 11). You will just have to update your software and Linux kernel version.
    Time Shift? Of course, frequent software changes may potentially lead to an unstable system. However, you can use Timeshift for backup.
    Supported Desktop Environment? Solus OS comes in many flavors of desktop environments, including Budgie (made by Solus developers), GNOME, MATE, and Plasma, so you don’t have to set them up yourself, again, unlike Arch Linux.
    User-friendlyness Windows and other Linux distro users will find using Solus OS familiar, as most GUI items are found in similar places, with a few tweaks to make it more user-friendly.
    FYI Moving your mouse over a category doesn’t immediately switch it like in other Linux distributions. Additionally, Solus Budgie has a nifty Windows-like menu on the side that can search installed applications on the left and a notification center, calendar, settings, and widgets to the right.
    Installation The installation process is effortless, similar to Linux Mint.
    Summary In conclusion, Solus OS is great for beginners and advanced users due to its inclusion of user-friendly features and the fact that it’s built from scratch, making it as customizable as Arch Linux but without the hassle.
