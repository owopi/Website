    What is POP OS?

    Pop OS

    		 Pop!_ OS is another free and open-source Linux distribution based on Ubuntu.
    Based on Ubuntu It uses apt to update, install, get repositories and packages! It can also use dpkg to open deb files! Pop!_OS features a customized GNOME Shell interface, with a Pop!_OS theme.
    Tailored made Desktop Environment! Pop!_OS features a custom desktop environment known as COSMIC, based on GNOME.
    Similarities Pop!_OS has a similar layout to macOS and Ubuntu, making switching between them effortless.

    What Makes Pop!_OS Shine?
    At first, Pop!_OS may seem like the average Ubuntu-based distribution, however, assuming so would be wrong.
    Pop!_OS offers lots of distinct productivity features. One of the most popular features is “auto-tiling”!
    This rearranges windows into tiles and gives you a bunch of keyboard shortcuts to navigate them.
    How does this feature help you, the user, or in general anyone using POP!_OS?
    This feature helps you spend less time navigating the OS and makes finding windows a breeze.
    Other features!
    Pop!_OS also has a bunch of trackpad gestures built-in to make navigation almost as fast as a mouse on laptops.
