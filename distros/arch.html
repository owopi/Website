    Arch Linux

    Advantages of Arch Linux

    Lightweight

    		 Arch Linux is built from scratch from the Linux kernel and not Ubuntu or Debian, like most popular distributions. Arch Linux lets users customize everything while keeping the functionality untouched.
    Complexity The complexity of Arch Linux forces you to learn things you would’ve never thought of in other distributions. For example, configuring the network while installing Arch Linux is a good learning lesson as it’s an extremely long process.
    Where to find guidance? You aren’t alone in the dark as people have anticipated and prepared Arch Wiki is there for you! It is the most extensive community-managed Linux documentation that exists on the internet.
    Customization extents? Arch Linux takes a DIY approach to a Linux distrubution by giving you the power to customize every intricate detail yourself! This makes Arch Linux Lightweight and accesible as there are no bloatwares.
    Disadvantages for rookies? It doesn't come with its own native desktop environment or window manager, meaning that you’ll have to set it up yourself; however, this gives you many options to choose from.

    Benefits
    Summary: Arch Linux is ightweight, complex, and customizable to your liking!

    AUR?

    Arch Linux is a rolling release distribution, meaning that you get new kernel and application versions as soon as they are released.

    Due to Arch Linux being so barebones compared to other distributions, it is swift to provide you with the latest kernel.

    In comparison, most other Linux distributions serve you old Linux kernel versions because the whole OS is built on top of the old version. The same goes for software.

    Because of the Arch User Repository (AUR), Arch Linux users get new software updates before other Linux users.
