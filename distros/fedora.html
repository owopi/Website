    Fedora? The Hat?

    It’s not Actually a Hat

    What is Fedora Linux Fedora Linux is another Linux distribution developed by the Fedora Project, sponsored by Red Hat and sponsored by other organizations.
    Potential Customers? Fedora is made for use in workstations, servers, and IoT applications.
    Fedora, the Hat? Do not be deceived, Fedora is an operating system and not a hat people wear! It is still not as well known yet!
    Open Source It is free and open-source to distribute and use commercially, like Oracle Linux.
    Desktop Environment Unlike other Red Hat-based distributions, Fedora has a desktop flavor with the GNOME desktop environment.
    Similarities It is prevalent for use among developer workstations and system admin's control centers. Fedora Workstation shines in productivity as it ships with new development and system administration tools straight from Red Hat.
    Server Quality Fedora Server doesn't make for a lousy server distribution; however, the infrequency of updates makes it unideal for large deployments, although it is still perfectly usable for testing.
    Disadvantages Not having an edge over others like mint linux has over ubuntu with it’s quality of life improvement and apps. It is not as well-known as other distributions as it doesn’t provide a clear advantage to an average Linux user; it primarily appeals to developers and administrators.
